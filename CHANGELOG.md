# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-22

### Added

- Initial release
- **File** resource: Get All, Get, Get Batch, Download operations
- **Folder** resource: Get All operation
- **AI** resource: Get Status, Get Notes, Get Questions operations
- **Share** resource: Get Private, Get Public operations
- Bearer token authentication
- CI workflow (lint + build)
